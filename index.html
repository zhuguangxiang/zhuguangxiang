Introduce

Koala is a fast, flexible and modularized modern language.


Hello World

Koala supports interactive, compile and run mode. This is 'Hello World' in interactive mode. 

    [james@host ~]$ koala
    koala 0.9.1
    [GCC 8.3.0] on Linux/x86_64 5.0.0-32-generic
    > "Hello, World"
    Hello World


Install

download source code

  • unzip koala-lang-master.zip
  • cd koala-lang-master
  • mkdir build
  • cd build
  • cmake ..
  • make
